# live2d模型预览

```sh
yarn add -D vuepress-plugin-live2d 
```

配置项：
```js
["vuepress-plugin-live2d",{
          "modelName": "koharu",
          "mobileShow": true,
          "position": "right" // 显示在左下角还是右下角}
      ],
```
