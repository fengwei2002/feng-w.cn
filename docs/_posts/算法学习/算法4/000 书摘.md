有些 Java 程序员用 int a[] 而不是 int[] a 来声明一个数组。这两者有什么不同？
答 在 Java 中，两者等价且都是合法的。前一种是 C 语言中数组的声明方式。后者是 Java 提倡的方式，
因为变量的类型 int[] 能更清楚地说明这是一个整型的数组。

在 Java 中，一个静态方法能够将另一个静态方法作为参数吗？
答 不行，但问得好，因为有很多语言都能够这么做。
