# vscode 使用技巧

以下是我多年使用的经验（虽然现在没用多久）

## 快捷键

[[ctrl+k]] + [[ctrl+s]] 打开快捷键一览表

[[ctrl+k]] + [[ctrl+t]] 召唤新主题

[[alt]]+[[shift]]+[[A]] 多行注释

[[ctrl]]+[[shift]]+[[s]] 另存为
